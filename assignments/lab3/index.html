作业说明
  • 使用 MNIST 数据集源码级实现朴素贝叶斯算法；
  • 阅读论文《An Analysis of Single­Layer Networks in Unsupervised Feature Learning》：
    • 基于上面的论文和本周所学内容，在 MNIST 数据集上源码级实现 PCA 降维和 K 均值聚类；
    • 将你的实现结果与使用 Scikit-Learn 框架的实现进行对比，尝试改进你的实现方法；
    • 撰写技术报告，对涉及到的原理和实现方法与结果进行简要解释(注意不是翻译论文)；
本周的作业初稿和终稿内容相同，大家有足够的时间在讨论课后进行内容改进。
注：本周开始讨论课改为学生上台进行演讲，请所有人提前做好相关准备。
注意事项
  • 技术报告不要求现在就能使用 LaTeX 语法完成，可使用 MarkDown 导出 PDF；
  • 可换用 Fashion-MNIST 数据集或是 Yale 数据集(鼓励大家随便玩各种不同的数据集)；
  • 可使用 DB-SCAN / ICA 等其它的聚类或者降维算法作为内容的补充；
