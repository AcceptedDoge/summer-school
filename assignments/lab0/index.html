常见问题
从清华镜像站下载 Anaconda 与配置 conda 和 pip 源(不然将从国外下载...)

Anaconda 安装包可以到 https://mirrors.tuna.tsinghua.edu.cn/anaconda/archive/ 下载。

TUNA 还提供了 Anaconda 仓库的镜像，运行以下命令:

conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main/
conda config --set show_channel_urls yes

pip install pip -U
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
即可添加 Anaconda Python 免费仓库。(国内还有除了清华以外的其它镜像站点)
如果你使用 pip 下载速度依旧很慢，则可以访问：
https://www.lfd.uci.edu/~gohlke/pythonlibs/
使用 Ctrl + F 寻找到对应的包，人为下载 whl 文件到本地进行安装。
在 Windows 右键菜单添加 “在此处打开 Anaconda 命令行” 功能

假设你的 Anaconda 安装路径为 D:\Anaconda\.

  1. Win + R 输入 regedit 后回车，打开注册表；
  2. 在 HKEY_CLASSES_ROOT\Directory\Background\shell\ 下新建一项默认值为 Anaconda Prompt Here，并新建项 command；
  3. 在 command 中默认值填入 cmd.exe /s /k "title Anaconda" && D:\Anaconda\Scripts\activate.bat D:\Anaconda (注意更改 Anaconda 路径)；
  4. 再在 Anaconda Prompt Here 下新建字符串值 Icon，数值数据为 D:\Anaconda\Menu\Iconleak-Atrous-Console.ico， 注意更改路径，这一步是为了能够在右键菜单显示对应的图标。
Jupyter Notebook 常用快捷键(提升使用效率) https://shortcutworld.com/Jupyter-Notebook/win/Jupyter-Notebook_Shortcuts (或谷歌一下)
比较常用的有 A / B / DD / Y / M / Shift + Enter / Alt+ Enter / ...
建议同时熟悉 Markdown 语法和 LaTeX 语法(网页端基于 MathJaX 使用).
相对路径找不到图片等本地资源文件

这是因为网站默认以单文件形式下载，所有练习文件可以在 这个链接 找到，有的时候 assets 文件夹中的文件(如图片) 以相对路径的形式进行了引用，这些文件则需要单独进行下载。你也可以使用一定的方法，仅下载 GitHub 存储库中某个子文件夹的内容。
